Personalized Walking Tour App

First iteration

This project aims at coding a personalized walking tour app for the city of Buenos Aires, aimed at tourists and leisure users. The design specification is as follows: the users will provide a list of interests freely written (or spoken), a time interval and (optionally) a point to which they want to arrive in the end. The app will suggest walking trips for the users based on their interests, their available time and the final destination.

Currently, the plan is to use the following technologies: Leaflet for the mapping, Leaflet Routing Machine and Mapboxfor the routing, Natural Language Processing for the users' interests analysis, speech processing for the users' interests recognition if spoken. This last feature could be implemented using the black-box models provided by AWS.

This first (very basic) iteration presents a map, asks for two pairs of coordinates from the user and shows a corresponding path.
